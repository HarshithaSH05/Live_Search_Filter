# 🛒 Creative Live Search Filter

A dynamic and interactive **Live Search Filter** project built using **HTML, CSS, and JavaScript (DOM)**.  
This project simulates real-world functionality where users can filter visible data instantly based on their input — commonly used in e-commerce platforms, contact lists, and dashboards.

---

## 📖 Project Definition
The project displays a list of items (products) along with a search input box.  
As the user types in the search field, the list dynamically updates to show only the items that match the entered text.  
Additionally, users can filter items by **category buttons** (Fruit, Berry, Tropical, Vegetable, All).

---

## 🎯 Problem Statement
Create a web page that:
- Displays a list of items (such as product names).
- Provides a search input field.
- Filters the list in **real-time** as the user types.
- Ensures the search is **case-insensitive**.
- Does not refresh the page while filtering.
- Displays **“No results found”** if no matching items are available.
- Allows filtering by **categories** for a more powerful search.

---

## ✅ Functional Requirements
- Display a list of items on the web page.
- Add a search input field.
- Filter the list in real-time as the user types.
- Ensure the search is case-insensitive.
- Do not refresh the page while filtering.
- Display “No results found” if no matching items are available.
- Provide category filter buttons for additional filtering.

---

## 🛠️ Technologies Used
- **HTML** → Structure of the page.
- **CSS** → Styling, animations, and layout.
- **JavaScript (DOM)** → Dynamic filtering, highlighting, and interactivity.

---

## 🌟 Features
- **Card-based UI** with product images and names.
- **Text search filter** (case-insensitive).
- **Category filter buttons** (Fruit, Berry, Tropical, Vegetable, All).
- **Highlighting** of matched text inside results.
- **Responsive grid layout** for modern dashboard feel.
- **Hover animations** for interactive cards.
- **“No results found”** message when nothing matches.
